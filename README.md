# QR Code Generator 🔳

This is a simple Python project that generates QR codes for different profile URLs such as LinkedIn, GitHub, portfolio, or any website link.

## 📌 Features

* Generate QR code for any URL
* User can enter multiple profile links
* Automatically saves QR codes as image files
* Easy and beginner-friendly Python project

## 🛠️ Technologies Used

* Python
* qrcode library
* PIL (Python Imaging Library)

## 📦 Installation

1. Clone the repository

git clone https://github.com/yourusername/qr-code-generator.git

2. Go to the project folder

cd qr-code-generator

3. Install required library

pip install qrcode[pil]

## ▶️ How to Run

Run the Python file:

python qr_generator.py

Then enter:

* Number of profiles
* Profile URL
* File name

The program will generate QR code images automatically.

## 📂 Example

Input:
Enter URL: https://linkedin.com/in/example

Output:
QR code image saved as linkedin.png
